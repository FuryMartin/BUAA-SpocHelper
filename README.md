## 📑 简介

Spoc 助手是一款基于 WinUI 3 的 Windows 10/11 应用，能够让 BUAAers 优雅地完成作业、查看课件

## 🥳 使用

- 🌐 通过微软商店下载本应用：[Spoc 助手](https://apps.microsoft.com/store/detail/9PM0GCZ6ZD53?launch=true&mode=full)
- 🐼 使用统一认证账号登录（你的账号信息将仅保存在本地😊）
- 🎉 查看作业并提交
- 🏠 查看并下载课件
